# Introduction

You can check site by clicking [https://kyrylorusakov.github.io/apple_gsap_threejs/](https://kyrylorusakov.github.io/apple_gsap_threejs/)

Recreate the Apple website, combining GSAP animations and Three.js 3D effects. From custom animations to animated 3D models.


## System Requirements

To get started with development, you need to install few tools

1. git 
   
   `git` version 2.13.1 or higher. Download [git](https://git-scm.com/downloads) if you don't have it already.

   To check your version of git, run:

   ```shell
    git --version
   ```

2. node 
   
   `node` version 16.15.1 or higher. Download [node](https://nodejs.org/en/download/) if you don't have it already.

   To check your version of node, run:

   ```shell
    node --version
   ```

3. npm
  
   `npm` version 5.6.1 or higher. You will have it after you install node.

   To check your version of npm, run:

   ```shell
    npm --version
   ```

## Setup

To set up a development environment, please follow these steps:

1. Clone the repo

   ```shell
    git clone https://github.com/KyryloRusakov/apple_gsap_threejs.git
   ```

2. Install the dependencies
   
    ```shell
     npm install
    ```

3. Run the app
   
    ```shell
    npm run dev
    ```

    Project will be running in the browser.

    Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
